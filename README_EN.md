# Factorio CPU

Programmable processor built on Factorio combinators without mods.

![screenshot]

## Characteristics
- 8 registers
- ~25 instructions
- 45 RAM cells per block (expandable)

## Quick Start
1. Place the CPU blueprint from the blueprint book
2. Write a program in the Listing
3. Send signal `[I]=1` to start
