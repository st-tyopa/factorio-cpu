# Factorio CPU

Программируемый процессор на комбинаторах Factorio без модов.

![скриншот общего вида]

## Характеристики
- 8 регистров
- ~25 команд
- 45 ячеек ОЗУ на блок (расширяемо)

## Быстрый старт
1. Установите чертёж CPU из книги чертежей
2. Напишите программу в Листинге
3. Подайте сигнал `[I]=1` для запуска

## Документация
Полная документация в [docs/documentation.md](docs/documentation.md)

## Применение
- Менеджер поездов
- Менеджер ресурсов
- Дашборд базы
- Любая задача с ветвистой логикой и циклами

## Совместимость
Factorio 2.0+, Space Age
